MULTI-ROUTER DEPLOYMENT GUIDE

To simulate multiple router nodes, deploy this container multiple times with custom metadata environment variables:

# Deploy Router 1 (US-East Edge)
gcloud run deploy router-us-east-01 --image=$REGION-docker.pkg.dev/$PROJECT_ID/cloud-run-source-deploy/router-emulator:latest \
  --set-env-vars="ROUTER_ID=RTR-US-EAST-01,ROUTER_LOCATION=New York DC,CONTROL_PASSWORD=PassEast123" --no-allow-unauthenticated

# Deploy Router 2 (EU-West Core)
gcloud run deploy router-eu-west-01 --image=$REGION-docker.pkg.dev/$PROJECT_ID/cloud-run-source-deploy/router-emulator:latest \
  --set-env-vars="ROUTER_ID=RTR-EU-WEST-01,ROUTER_LOCATION=London DC,CONTROL_PASSWORD=PassWest456" --no-allow-unauthenticated
